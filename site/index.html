MiniMask is a Chrome extension that gives Users a basic wallet AND Minima functionality that can be accessed from the webpage

The GitHub repo is hosted at https://github.com/minima-global/minimask

To install..

Download this zip file minimask-1.5.zip

Unzip this file into your Documents folder

Enable Developer mode in chrome://extensions

Use Load Unpacked from that page and select the extracted folder
