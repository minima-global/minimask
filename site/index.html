MiniMask is a Chrome extension that gives Users a basic wallet AND Minima functionality that can be accessed from a webpage

The GitHub repo is hosted at https://github.com/minima-global/minimask

To install..

Download this zip file minimask-2.5.zip

Unzip this file into your Documents folder

Enable Developer mode in chrome://extensions

Use Load Unpacked from that page and select the extracted folder

Once you have done that you can check out the examples on this site

Personally - I set the extension permission to enable when "You click on the extension"

Check out the Help section to see how you can USE MiniMask on your site
